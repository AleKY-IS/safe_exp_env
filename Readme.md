# Safe Exploration env

An [OpenAI safety-gym third-party environment](https://openai.com/blog/safety-gym/) with [SUMO](https://www.eclipse.org/sumo/). This repo is **modified** from the [original version](https://github.com/idthanm/env_build).

## System Requirements
- Ubuntu 16.04 or 18.04
- SUMO <= 1.7.0
- [safety-gym environment](https://github.com/openai/safety-gym)
